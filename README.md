# **江苏科技大学苏州理工学院2020上学期数据结构**

- 实验一 线性表及其应用

    1. SeqList:顺序表的基本操作
    2. LinkList:链表基本操作

- 实验二 栈和队列及其应用

    1. TransForm:进制转换
    2. YangHuiTriangle:杨辉三角
    3. Calculate:四则运算

- 实验三 串的操作
    1. CountNum:统计单词数
    2. SearchStr:搜索字符串，KMP、BF实现

- 实验四 二叉树及其应用
    1. BinaryTree:统计二叉树中叶子结点的个数
    2. HuffmanTree:用哈夫曼树对文本文档进行压缩和解压操作
